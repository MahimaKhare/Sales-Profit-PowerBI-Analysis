# Sales-Profit-PowerBI-Analysis

## Project Overview
This Power BI dashboard provides insights into sales, profit, and customer trends to support business decision-making.  
The dashboard includes KPIs, category analysis, top product insights, and state-level sales distribution.

## Key Insights
- Total sales and profit comparison
- Customer segmentation & region-wise performance
- Top performing and low performing categories
- Drill-down analysis with interactive visuals

## Tools Used
- Power BI
- DAX Measures
- Data Cleaning & Modeling
- Excel as data source (if applicable)

## Files Included
- `.pbix` file (Power BI dashboard)
- Screenshot folder of visuals
